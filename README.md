# GitHub Maker/Reviewer Web Todo

Disposable Vite + React + TypeScript app for aiops-platform GitHub
maker/reviewer end-to-end validation.

## Commands

- `npm ci`
- `npm test`
- `npm run build`
- `npm run test:e2e`

## Baseline

The seed app supports local todo creation, completion toggles, deletion, and
localStorage persistence. Feature issues extend this baseline through normal
GitHub PRs.
